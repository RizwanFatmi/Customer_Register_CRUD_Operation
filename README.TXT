

-------------------------------------------------------------------------------
                         Steps for installating the "Customer Register" 
-------------------------------------------------------------------------------


  1.  Install JDK-18(Java Development Kit)

  2.  Extract the Customer.Register RAR file

  3.  Open "Customer_Register" folder.	

  4.  Click on  the "Customer_Register" jar file to open the application.

  5.  Enjoy.........!!
